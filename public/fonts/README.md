# Brand fonts (public/fonts)

| File | Role |
|------|------|
| `Barnule-Regular.ttf` (or `.otf`) | Logo / brand wordmark |
| `Flexing-Regular.ttf` (or `.otf`) | Headlines |
| `Avona-Regular.ttf` | Body text (**already installed**) |

## Status
- **Avona** — downloaded and saved as `Avona-Regular.ttf`
- **Barnule** / **Flexing** — add manually (trial downloads from third-party hosts were blocked in this environment)

## Where to get them (personal-use trials)
1. **Barnule** — [Zarma Type](https://zarmatype.com/font/barnule-mauren-western-font-duo/) or FontSpace “Barnule & Mauren” trial  
2. **Flexing** — [1001 Fonts Flexing Demo](https://www.1001fonts.com/flexing-demo-font.html)  
3. Rename files exactly as in the table above and put them in this folder

Commercial products need paid licenses from the foundries.

Then restart `npm run dev`.
